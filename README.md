# Projet_Application_De_Restauration_Maison_Des_Saveurs
MaisonDesSaveurs est une appli de commande de nourriture avec paiement en ligne et inscription via Instagram/Facebook. Elle intègre un assistant IA (FastAPI) pour aider à choisir les plats, voir les promotions, suivre les commandes et répondre aux questions, offrant une expérience rapide et personnalisée.
